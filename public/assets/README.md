# Video Assets

## Adding Your Video

Place your video file here as `brain-video.mp4`

### Recommended Specifications:
- **Format**: MP4 (H.264 codec)
- **Resolution**: 1080x1080 or higher (square aspect ratio)
- **Duration**: 2-5 seconds (seamless loop)
- **File Size**: < 5MB recommended
- **Content**: 3D rendered brain/sphere animation

### Quick Test Video

If you don't have a video yet, the interface will:
1. Try to load `/assets/brain-video.mp4` (this file)
2. Fall back to a demo video from the internet
3. Show a beautiful animated CSS sphere if all videos fail

### Creating a Test Video

You can create a simple test video using:
- **Online tools**: Canva, Kapwing, or similar
- **3D Software**: Blender, Cinema 4D
- **Screen recording**: Record a loop of an animated gradient
